Документация профиля Woodmart

На этой странице собраны справочные сведения: структура JSON, пояснения к полям и готовый пример импорта. Материал предназначен для дизайнеров, контент-редакторов и ИИ-помощников, которые должны собирать корректный конфиг для темы Woodmart.

Структура JSON

Редактор генерирует файл вида:

{
  "name": "Название профиля",
  "generatedAt": "2024-05-01T12:00:00.000Z",
  "woodmart": {
    "header": { "color": "rgb(…)" },
    "footer": { "color": "rgb(…)" },
    "typography": {
      "basic": { "fontFamily": "…", "style": "…", "type": "…", "fontSize": "…", "color": "rgb(…)" },
      "title": { "fontFamily": "…", "style": "…", "textTransform": "…", "color": "rgb(…)" },
      "entities": { "fontFamily": "…", "hoverColor": "rgb(…)" },
      "secondary": { "fontFamily": "…" },
      "widget": { "fontFamily": "…", "fontSize": "…", "color": "rgb(…)" },
      "header": { "fontFamily": "…", "fontSize": "…" }
    },
    "colors": { "primary": "rgb(…)", "secondary": "rgb(…)" },
    "backgrounds": {
      "pages": { "all": "rgb(…)" },
      "shop": { "archive": "rgb(…)", "single": "rgb(…)" },
      "blog": { "archive": "rgb(…)", "single": "rgb(…)" }
    },
    "buttons": {
      "default": { "fontFamily": "…", "background": "rgb(…)", "hoverBackground": "rgb(…)" },
      "accent": { "fontFamily": "…", "background": "rgb(…)", "hoverBackground": "rgb(…)" }
    },
    "links": {
      "color": { "regular": "rgb(…)", "hover": "rgb(…)" },
      "underline": { "regular": "solid", "hover": "solid" },
      "underlineColor": { "regular": "rgb(…)", "hover": "rgb(…)" }
    }
  }
}

Все поля поддерживают формат rgb(255, 255, 255) и HEX. Если значение не задано, Woodmart возьмёт базовое из блока Basic.

Пояснения к разделам

Header и Footer

Определяют фон шапки и подвала. Следите за контрастом с логотипом и текстами меню (не ниже 4.5:1).

Typography

Basic заполняет весь текст по умолчанию; остальные разделы переопределяют ключевые зоны: заголовки, карточки, меню и виджеты. Если шрифт не указан, используется Basic.

Colors

Primary управляет кнопками и акцентами, Secondary — дополнительными блоками в конструкторе страниц.

Backgrounds

Позволяют разделять разделы сайта: общий фон, магазин, блог. Используйте мягкие контрасты, чтобы карточки не терялись.

Buttons

Default — стандартные кнопки темы. Accent пригодится для промо-блоков или альтернативных CTA.

Links

Цвета и оформление подчёркивания для обычного и hover-состояний. Для ясности фиксируйте и стиль, и цвет подчёркивания.

Готовый пример профиля

В репозитории лежит файл profiles/sample-profile.json — эталон для импорта и тестирования. Он синхронизирован с текущей версией редактора.

Чтобы быстро поделиться конфигом с командой или ИИ:

  1. Откройте вкладку «Woodmart».
  2. Нажмите «Скопировать JSON» либо скачайте файл.
  3. Передайте JSON в чат/сервис. Он уже содержит всю структуру, описанную выше.
Если нужен дополнительный пресет — соберите его в редакторе и экспортируйте JSON. Страница обновляется автоматически.
